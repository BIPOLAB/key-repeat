Key Repeat is a free beat repeat sampler.
Use it to record note repeats and stutters in a DAW.

FEATURES

  • Simple and intuitive.

    Drag in any audio sample and Key Repeat automatically syncs to your DAW’s tempo. Creating hi-hat patterns, snare rolls, and stutters has never been easier.

  • Keyswitch controlled repeats.

    Key Repeat lets you adjust repeat rate on the fly using keyswitches. Although they default to C0 thru F#0, they can be modified to suit the size of your keyboard.

  • Lightweight yet customizable.

    Key Repeat leaves a minimal memory/CPU footprint, yet comes with a complete set of controls. Try our Latch and EZ Triplet modes, and pick the one that’s most intuitive for you!

DOWNLOAD

Key Repeat is free and open source. The plugin is available in VST3 and AU formats, and must be run in a digital audio workstation (DAW).

We support Windows 7/8/10 and macOS Sierra or newer.
