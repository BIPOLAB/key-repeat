Key Repeat is a free beat repeat sampler.
Use it to record note repeats and stutters in a DAW.

FEATURES

Simple and intuitivess

Drag in any audio sample and Key Repeat automatically syncs to your DAWâ€™s tempo. Creating hi-hat patterns, snare rolls, and stutters has never been easier.

Keyswitch controlled repeats

Key Repeat lets you adjust repeat rate on the fly using keyswitches. You can even modify the keyswitch octave to suit the size of your keyboard.

Lightweight yet customizable

Key Repeat uses minimal CPU and memory resources, yet comes with a complete set of controls. Try on our Latch and EZ Triplet modes for size!

DOWNLOAD

Key Repeat is free and open source. The plugin is available in VST3 and AU formats, and must be run in a digital audio workstation (DAW).

We support Windows 7/8/10 and macOS Sierra or newer.
